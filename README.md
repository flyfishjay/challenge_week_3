# Crypto Arbitrage 
This is a python built analysis to see if there are any arbitrage opportunities across different crypto exchanges.  Roughly 3 months of data are used 
with trades being captured every minute.

## Technologies

Pandas 1.2.4
matplotlib 3.3.4
